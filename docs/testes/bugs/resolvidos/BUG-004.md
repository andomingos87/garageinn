# BUG-004: Admin não consegue excluir chamado

## Informações
- **Data:** 15/01/2026
- **Teste:** ADM-017
- **Usuário Testado:** Administrador (admin@garageinn.com.br)
- **Executor:** GPT-5.2 Codex

## Comportamento Esperado
Admin deve conseguir **excluir** um chamado existente.

## Comportamento Atual
Na tela do chamado não existe opção/botão para **Excluir**.

## Passos para Reproduzir
1. Fazer login como admin.
2. Abrir qualquer chamado (ex: #17).
3. Verificar a área de ações.

## Evidências
- Screenshot: `.playwright-mcp/bug-004-adm-017-excluir-chamado-sem-botao.png`

## Observações
Somente botões "Iniciar Cotação", "Negar" e "Cancelar" aparecem.

---

## Resolução

**Status:** ✅ Corrigido  
**Data:** 17/01/2026

### Causa Raiz

O botão de exclusão de chamados não estava sendo exibido nas páginas de detalhes dos chamados para usuários com permissão `admin:all`.

### Correção Aplicada

Criado novo componente `delete-ticket-button.tsx` e integrado às páginas de detalhes de chamados:

- `apps/web/src/app/(app)/chamados/components/delete-ticket-button.tsx` - Novo componente reutilizável
- `apps/web/src/app/(app)/chamados/compras/[ticketId]/page.tsx` - Integração do botão
- `apps/web/src/app/(app)/chamados/manutencao/[ticketId]/page.tsx` - Integração do botão
- `apps/web/src/app/(app)/chamados/rh/[ticketId]/page.tsx` - Integração do botão
- `apps/web/src/app/(app)/chamados/sinistros/[ticketId]/page.tsx` - Integração do botão

### Validação

Testado em 17/01/2026:
- Login como admin@garageinn.com.br
- Acessar Chamados → Chamado #15 "Compra de DDD"
- Botão "Excluir Chamado" visível na área de ações
