# BUG-001: Manobrista vê menu Unidades

## Informações
- **Data:** 15/01/2026
- **Teste:** OPR-MAN-N01
- **Usuário Testado:** Manobrista (Operações) - manobrista_operacoes_teste@garageinn.com
- **Executor:** GPT-5.2 Codex

## Comportamento Esperado
O menu **Unidades** não deve estar visível para Manobrista ou o acesso deve ser negado com mensagem apropriada.

## Comportamento Atual
O menu **Unidades** aparece na lateral e o acesso a `/unidades` é permitido, sem bloqueio explícito (página abre sem mensagem de acesso negado).

## Passos para Reproduzir
1. Fazer login como admin (`admin@garageinn.com.br`).
2. Ir em `Usuários` e personificar **Teste Manobrista - Operações**.
3. Verificar o menu lateral.
4. Acessar `/unidades`.
