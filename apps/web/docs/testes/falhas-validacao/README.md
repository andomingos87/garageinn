# Falhas de Validacao

Esta pasta armazena registros de falhas encontradas durante a validacao
das fases do backlog. Nao corrigir falhas aqui; apenas documentar.

## Padrao de registro

Crie um arquivo por fase, por exemplo:
- `FASE-0.md`
- `FASE-1.md`
- `FASE-2.md`
- `FASE-3.md`
- `FASE-4.md`
- `FASE-5.md`

## Template de entrada

- **Tarefa**: (copiar do `projeto/BACKLOG.md`)
- **Contexto**: (modulo/tela/fluxo)
- **Ambiente**: (local/dev/staging, data, commit)
- **Passos para reproduzir**: (numerado)
- **Resultado esperado**: (criterio de aceite)
- **Resultado obtido**: (erro/print/trace)
- **Evidencias**: (links para screenshot/trace/log)
- **Severidade**: (baixa/media/alta)
- **Observacoes**: (impacto, hipoteses)
