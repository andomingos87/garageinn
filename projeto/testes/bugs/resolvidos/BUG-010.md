- **Teste:** OPR-SUP-009
- **Usuário Testado:** Supervisor (Operações) - supervisor_operacoes_teste@garageinn.com

Supervisor deve acessar a tela de Unidades e visualizar dados das unidades de cobertura.

## Comportamento Atual
Ao acessar `/unidades`, a aplicação redireciona para o Dashboard sem exibir a listagem.

1. Fazer login como admin.
2. Personificar **Teste Supervisor - Operações**.
3. Tentar acessar `Unidades` pelo menu ou via URL `/unidades`.
