# BUG-009: Encarregado não consegue acessar Unidades

## Informações
- **Data:** 15/01/2026
- **Teste:** OPR-ENC-009
- **Usuário Testado:** Encarregado (Operações) - encarregado_operacoes_teste@garageinn.com
- **Executor:** GPT-5.2 Codex

## Comportamento Esperado
Encarregado deve conseguir acessar a tela de Unidades e visualizar dados básicos da unidade.

## Comportamento Atual
Ao acessar `/unidades`, a aplicação redireciona para o Dashboard sem exibir a listagem de unidades.

## Passos para Reproduzir
1. Fazer login como admin.
2. Personificar **Teste Encarregado - Operações**.
3. Tentar acessar `Unidades` pelo menu ou via URL `/unidades`.

- Screenshot: `.playwright-mcp/bug-009-encarregado-sem-acesso-unidades.png`

## Observações
O menu exibe "Unidades", mas a página não carrega.
